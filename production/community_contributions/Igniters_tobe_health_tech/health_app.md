# Project Details

## Links

- Repo: https://github.com/zeemanUka/Health_Product
- Live app: https://health-product-lime.vercel.app/

## Features

- Clinician sign-in with Clerk
- Protected app pages
- Patient list and patient search
- Patient create and edit flow
- Tag support for patients
- Consultation note form
- Voice dictation for raw notes
- AI note transform with OpenRouter
- Patient email draft generation
- Email sending with SMTP
- Profile page for clinician details
- Settings page for sender details and app controls
- Placeholder subscription gate for protected routes
- FastAPI backend with REST endpoints
- SQLite database with Alembic migrations
- Backend tests for core flows
